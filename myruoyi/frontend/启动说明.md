# API全生命周期管理平台前端启动说明

## 环境要求

- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

## 快速启动

### Windows用户

1. 双击运行 `start.bat` 文件
2. 或者在命令行中执行：
   ```bash
   cd frontend
   npm run dev
   ```

### Mac/Linux用户

```bash
cd frontend
npm install
npm run dev
```

## 访问地址

启动成功后，在浏览器中访问：http://localhost:3000

## 功能模块

1. **用户认证**
   - 登录/登出功能
   - JWT Token管理
   - 用户信息存储

2. **API应用管理**
   - 应用列表查询
   - 应用增删改查
   - 状态管理

3. **API模块管理**
   - 树形结构展示
   - 模块增删改查
   - 层级关系管理

4. **API接口管理**
   - 接口列表查询
   - 接口详细信息管理
   - 请求/响应参数配置
   - 发布状态控制

5. **工作台**
   - 数据统计展示
   - 最近创建记录

## 注意事项

1. 确保后端服务已启动（默认端口8080）
2. 前端默认代理到后端API地址
3. 首次运行需要安装依赖：`npm install`
4. 如遇到端口冲突，可修改 `vite.config.js` 中的端口配置

## 常见问题

### 依赖安装失败
```bash
# 清除缓存
npm cache clean --force

# 删除node_modules
rm -rf node_modules

# 重新安装
npm install
```

### 端口被占用
```bash
# 查看端口占用
netstat -ano | findstr :3000

# 杀死进程
taskkill /F /IM node.exe
```

### API请求失败
1. 检查后端服务是否正常启动
2. 检查网络连接
3. 查看浏览器控制台错误信息

## 开发建议

1. 修改代码后及时刷新浏览器查看效果
